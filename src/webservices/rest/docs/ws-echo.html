WS Echo Service

This service echoes back the message provided by the client.

Endpoint

POST /webservices/rest/ws-echo.php

Request Parameters

  • message (string): The message to be echoed back.

Example Burp Repeater Request

POST /webservices/rest/ws-echo.php HTTP/1.1
Host: mutillidae.localhost
Accept: application/json
Content-Type: application/x-www-form-urlencoded
Origin: http://mutillidae.localhost
Connection: close
Content-Length: 20

message=Hello+World!

cURL Example

curl -X POST "http://mutillidae.localhost/webservices/rest/ws-echo.php" \
 -H "Host: mutillidae.localhost" \
 -H "Accept: application/json" \
 -H "Origin: http://mutillidae.localhost" \
 -H "Content-Type: application/x-www-form-urlencoded" \
 --data "message=Hello+World!"

Response

{
    "message": "Hello World!",
    "command": "echo 'Hello World!'",
    "security-level": 0,
    "result": "Hello World!"
}
