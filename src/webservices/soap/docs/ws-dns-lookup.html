SOAP DNS Lookup Service Documentation

The SOAP DNS Lookup Service allows users to perform DNS lookups through a SOAP interface.

Endpoint

POST /soap/ws-dns-lookup.php

Request Structure

<soapenv:Envelope xmlns:soapenv="http://schemas.xmlsoap.org/soap/envelope/">
   <soapenv:Header/>
   <soapenv:Body>
      <dnsLookup>
         <hostname>example.com</hostname>
      </dnsLookup>
   </soapenv:Body>
</soapenv:Envelope>

Example Using curl

curl -X POST "http://mutillidae.localhost/soap/ws-dns-lookup.php" \
 -H "Content-Type: text/xml; charset=utf-8" \
 -H "SOAPAction: \"urn:dnsLookup\"" \
 --data '<soapenv:Envelope xmlns:soapenv="http://schemas.xmlsoap.org/soap/envelope/">
   <soapenv:Header/>
   <soapenv:Body>
      <dnsLookup>
         <hostname>example.com</hostname>
      </dnsLookup>
   </soapenv:Body>
</soapenv:Envelope>'

Expected Response

<?xml version="1.0"?>
<soap:Envelope xmlns:soap="http://schemas.xmlsoap.org/soap/envelope/">
   <soap:Body>
      <dnsLookupResponse>
         <ipAddress>93.184.216.34</ipAddress>
      </dnsLookupResponse>
   </soap:Body>
</soap:Envelope>
