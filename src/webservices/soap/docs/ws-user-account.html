SOAP User Account Service Documentation

The SOAP User Account Service allows users to create, retrieve, update, and delete user accounts via SOAP requests.

Endpoint

POST /soap/ws-user-account.php

Request Structure - Create User

<soapenv:Envelope xmlns:soapenv="http://schemas.xmlsoap.org/soap/envelope/">
   <soapenv:Header/>
   <soapenv:Body>
      <createUser>
         <username>john</username>
         <password>12345</password>
         <firstname>John</firstname>
         <lastname>Doe</lastname>
         <signature>JDoe</signature>
      </createUser>
   </soapenv:Body>
</soapenv:Envelope>

Example Using curl - Create User

curl -X POST "http://mutillidae.localhost/soap/ws-user-account.php" \
 -H "Content-Type: text/xml; charset=utf-8" \
 -H "SOAPAction: \"urn:createUser\"" \
 --data '<soapenv:Envelope xmlns:soapenv="http://schemas.xmlsoap.org/soap/envelope/">
   <soapenv:Header/>
   <soapenv:Body>
      <createUser>
         <username>john</username>
         <password>12345</password>
         <firstname>John</firstname>
         <lastname>Doe</lastname>
         <signature>JDoe</signature>
      </createUser>
   </soapenv:Body>
</soapenv:Envelope>'

Expected Response

<?xml version="1.0"?>
<soap:Envelope xmlns:soap="http://schemas.xmlsoap.org/soap/envelope/">
   <soap:Body>
      <createUserResponse>
         <message>User created successfully.</message>
      </createUserResponse>
   </soap:Body>
</soap:Envelope>

Other Operations

  • Retrieve User: Use the getUser operation with the username as input.
  • Update User: Use the updateUser operation with the updated fields.
  • Delete User: Use the deleteUser operation with the username.

Troubleshooting

Troubleshooting Tips:
  • Ensure the SOAPAction header matches the operation.
  • Check that all required fields are included in the request.
  • Make sure your server is properly configured to accept SOAP requests.
